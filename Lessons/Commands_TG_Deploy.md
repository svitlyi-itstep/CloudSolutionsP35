# Корисні команди консолі EC2

## Оновлення системи та встановлення залежностей

| Команда | Опис |
|---------|------|
| `sudo yum update -y` | Оновити всі системні пакети (Amazon Linux) |
| `sudo yum install git -y` | Встановити Git |
| `sudo yum install python3 -y` | Встановити Python 3 |
| `sudo yum install python3-pip -y` | Встановити pip |
| `python3 --version` | Перевірити версію Python |
| `git --version` | Перевірити версію Git |

---

## Робота з Git

| Команда | Опис |
|---------|------|
| `git clone https://github.com/svitlyi-itstep/P35TgBot.git` | Клонувати репозиторій на сервер |
| `cd P35TgBot` | Перейти до директорії проєкту |
| `git pull` | Отримати останні зміни з репозиторію |
| `git status` | Переглянути стан робочої директорії |
| `git log --oneline -5` | Переглянути останні 5 комітів |

## Налаштування Python-середовища

| Команда | Опис |
|---------|------|
| `python3 -m venv .venv` | Створити віртуальне середовище |
| `source .venv/bin/activate` | Активувати віртуальне середовище |
| `deactivate` | Деактивувати віртуальне середовище |
| `pip install -r requirements.txt` | Встановити всі залежності з файлу |
| `pip list` | Переглянути встановлені пакети |

---

## Налаштування змінних оточення

| Команда | Опис |
|---------|------|
| `cp .env-example .env` | Створити файл `.env` з шаблону |

## Робота з screen (автономна робота бота)

`screen` дозволяє боту продовжувати роботу після закриття SSH-сесії.

| Команда | Опис |
|---------|------|
| `sudo yum install screen -y` | Встановити screen |
| `screen -S bot` | Створити нову сесію з іменем `bot` |
| `screen -ls` | Переглянути список активних сесій |
| `screen -r bot` | Повернутися до сесії з іменем `bot` |
| `screen -r <ID>` | Повернутися до сесії за її ID |
| `Ctrl+A, D` | Від'єднатися від сесії (бот продовжує працювати) |
| `Ctrl+A, K` | Завершити поточну сесію screen |
| `Ctrl+C` | Зупинити бота всередині сесії screen |

## Типовий сценарій запуску через screen

```bash
screen -S bot                   # 1. Створити сесію
source .venv/bin/activate       # 2. Активувати venv
python3 bot.py                  # 3. Запустити бота
# Ctrl+A, D                     # 4. Від'єднатися — бот працює у фоні
```
